Tag difference
ชื่อแทร็ค ความเหมือน ความแตกต่าง
Tag hr ขึ้นบรรทัดใหม่ ขีดเส้น
Tag br ขึ้นบรรทัดใหม่ ไม่ขีดเส้น
Tag ul กำหนดรายการ กำหนดรายการที่ไม่มีลำดับ
Tag ol กำหนดรายการ กำหนดรายการที่มีลำดับ
Tag chekboxes กำหนดทางเลือก กล่องเลือกแบบหลายทาง
Tag select กำหนดทางเลือก กำหนดรายการที่ให้เลือก(แบบ Drop-down)
Tag radio button กำหนดทางเลือก กำหนดให้ผู้ใช้งานสามารถได้เพียงตัวเลือกเดียวจากชุดของตัวเลือกทั้งหมด
tag div กล่องข้อความ กล่องข้อความ อยู่บรรทัดถัดไปจาก Element ก่อนหน้า
tag กล่องข้อความ กล่องข้อความ อยู่บรรทัดเดียวกัน Element ก่อนหน้า
tag script กำหนดข้อมูล สคริปต์ฝั่งไคลเอ็นต์ มีการจัดการภาพ, ตรวจสอบรูปแบบและการเปลี่ยนแปลงแบบไดนามิกของเนื้อหา
tag Style กำหนดข้อมูล สำหรับรูปแบบเอกสาร HTML
